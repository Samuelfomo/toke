🚀 TOKÉ - Liste des Fonctionnalités à Implémenter

🎯 CONCEPT GÉNÉRAL

  • Vision : Transformer le smartphone en badgeuse universelle
  • Principe : Gestion par sessions de travail (Clock-in → Clock-out)
  • Sécurité : QR Code + GPS obligatoires avec système anti-fraude

👥 GESTION DES UTILISATEURS

🔐 Authentification Multi-Niveaux

  • OTP Onboarding : Code 6 chiffres pour première connexion
  • PIN Employé : 4 chiffres + biométrie pour pointage quotidien
  • Password Manager : Email + mot de passe pour gestion équipe

🏗️ Système de Rôles Flexibles

  • EMPLOYEE : Pointage personnel, création mémos, consultation historique
  • MANAGER : + Gestion équipe, validation mémos, création sites/QR
  • MANAGER_SENIOR : + Supervision managers, escalade, analytics
  • HR_ADMIN : Accès tenant complet, gestion utilisateurs, config système
  • AUDITOR : Pointage multi-sites, rapports audit, missions temporaires
  • Multi-rôles : Combinaisons possibles (ex: Manager qui pointe aussi)

🏢 Hiérarchie Organisationnelle

  • Structure N-niveaux : Hiérarchie flexible et évolutive
  • Relations superviseur-subordonné : Avec dates d’effectivité
  • Délégations : Gestion missions temporaires et intérims

📱 MODULE POINTAGE CORE

⏰ Types de Pointage

  • clock_in : Ouverture session de travail
  • clock_out : Fermeture session de travail
  • pause_start : Début pause (déjeuner, etc.)
  • pause_end : Fin de pause
  • external_mission : Mission externe au site principal

🏢 Gestion Sites Flexible

  • Manager Sites : QR personnel du manager pour son équipe
  • Global Sites : Sites partagés (bureaux, agences communes)
  • Temporary Sites : Missions courtes, chantiers temporaires
  • Public Sites : Lieux publics (gares, aéroports) pour commerciaux/auditeurs

🗺️ Géofencing Intelligent

  • Polygones complexes : Zones géographiques précises par site
  • Validation obligatoire : GPS + QR Code pour chaque pointage
  • Flexibilité missions : Assouplissement GPS pour missions externes

🔄 MODULE SYNCHRONISATION OFFLINE

📱 Architecture Offline-First

  • Stockage local SQLite : Pointages en attente de synchronisation
  • Fonctionnement 7 jours : Sans connexion réseau
  • Sync intelligente : Upload automatique à la reconnexion
  • Auto-retry : Tentatives multiples avec backoff exponentiel

⚔️ Gestion Conflits

  • Détection automatique : Conflits entre données locale/serveur
  • Résolution auto : Si écart < 30min
  • Escalade manager : Pour conflits complexes
  • Timestamp serveur : Fait autorité en cas de conflit

📝 MODULE MÉMOS & JUSTIFICATIONS

📋 Types de Mémos

  • Préventifs : Anticipation retard/absence (“Demain rdv médical”)
  • Correctifs : Justification après incident (“Oubli pointage hier”)
  • Auto-générés : Système détecte anomalie et crée mémo automatique

✅ Workflow Validation

  • Délai 24h : Manager a 24h pour valider/rejeter
  • Escalade automatique : Si délai dépassé, remonte au N+1
  • Statuts : draft → submitted → pending → approved/rejected

📎 Pièces Jointes

  • Photos : Justificatifs, tickets transport (max 5MB)
  • Audio : Explications vocales 30s max
  • Métadonnées : GPS, device info, timestamp légal

🚨 MODULE ANTI-FRAUDE

🔍 Détections Automatiques

  • Géofencing violation : Pointage hors zone autorisée
  • Doublons temporels : Même site/employé < 15min
  • Vitesse impossible : Déplacements irréalistes entre sites
  • GPS spoofing : Détection mock location Android
  • Patterns suspects : Comportements récurrents anormaux

⚠️ Niveaux d’Alerte

  • Low : Logs système uniquement
  • Medium : Notification manager différée
  • High : Notification manager immédiate
  • Critical : Blocage pointage + investigation manuelle

📊 FONCTIONNALITÉS SESSIONS DE TRAVAIL

🕐 Gestion Sessions

  • Session unique : Un employé = une session active maximum
  • Calculs automatiques : Durée travail/pause par session
  • Sessions chevauchantes : Gestion équipes de nuit (jour J → jour J+1)
  • Fermeture automatique : Si oubli détecté au prochain pointage

📈 Calculs & Rapports

  • Durées automatiques : Temps travaillé/pause calculé en temps réel
  • Rapports journaliers : Envoyés automatiquement aux managers
  • Historique complet : Consultation données par employé/période
  • Export paie : Données formatées pour SIRH externes
